\section{Core Package (\texttt{src/core/})}
\label{sec:core}
% ─────────────────────────────────────────────────────────────────────────────

The core package provides shared primitive types, configuration loading, CLI argument
parsing, and the seeded random-number generator used throughout the simulation.

\subsection{\texttt{Vec2} (struct)}
\label{sec:vec2}

Plain aggregate for 2-D floating-point coordinates.  Chosen as a struct (no encapsulation
overhead) because it appears in tight physics loops.

\texttt{AgentId} is a type alias: \texttt{using AgentId = std::uint32\_t}.  It uniquely
identifies each agent within a generation.

% ─────────────────────────────────────────────────────────────────────────────
\subsection{\texttt{BoundaryMode} (enum class)}

\begin{lstlisting}
enum class BoundaryMode { Clamp, Wrap };
\end{lstlisting}

\texttt{Clamp} constrains positions to $[0,w]\times[0,h]$.
\texttt{Wrap} applies toroidal topology via \texttt{std::fmod}.
Controlled by the \texttt{"boundary\_mode"} field in \texttt{default\_config.json}.

\subsection{\texttt{Random} (class)}

Single 64-bit Mersenne Twister instance shared by simulation, evolution, and data layers.
A single instance per simulation ensures reproducible runs from any given seed.

\begin{table}[H]
\centering\small

\begin{itemize}[nosep]

\end{itemize}
